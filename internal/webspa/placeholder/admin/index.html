Herold admin

Admin SPA not yet built. From a clean checkout:

make build-web
make build

This runs pnpm -C web install --frozen-lockfile, pnpm -C web build, and copies the resulting web/apps/admin/dist/ into this binary's embedded FS. The Svelte admin app source lives at web/apps/admin/. For development hot-reload without rebuilding the binary, run pnpm --dir web --filter @herold/admin dev which starts a Vite dev server on port 5174 proxying the herold admin listener.
